# R07 — Registro de Discipulado

Una aplicación web interactiva diseñada para el seguimiento y registro de actividades de discipulado, organizada visualmente a través de un calendario dinámico.

## 🚀 Características

- **Calendario Dinámico:** Visualización por meses con navegación fluida.
- **Registro Detallado:** Modal interactivo para registrar actividades diarias.
- **Persistencia Local:** Los datos se guardan automáticamente para no perder el progreso.
- **Optimizado para Impresión:** Incluye estilos específicos para generar reportes en PDF o imprimir el registro mensual.
- **Interfaz Moderna:** Diseño limpio con CSS personalizado y experiencia de usuario intuitiva.

## 🛠️ Tecnologías

- **HTML5:** Estructura semántica.
- **CSS3:** Diseño responsivo y estilos de impresión.
- **JavaScript (Vanilla):** Lógica del calendario, manejo de eventos y almacenamiento local.

## 📂 Estructura del Proyecto

- `index.html`: Estructura principal de la aplicación.
- `style.css`: Estilos visuales y configuraciones para impresión.
- `app.js`: Lógica de funcionamiento, gestión de fechas y persistencia.
- `.gitignore`: Configuración para excluir archivos temporales y de configuración local.

## 📥 Instalación y Uso

1. Clona este repositorio o descarga los archivos.
2. Abre `index.html` en cualquier navegador web moderno.
3. ¡Comienza a registrar tu progreso!
